Included Files:

1. create_tables.sql
   - Creates all necessary tables (Customer, Product, Order, etc.) with appropriate constraints.

2. insert_data.sql
   - Populates the database with test data including 20+ products, 5 customers, and at least one order per customer.

3. drop_tables.sql
   - Drops all tables in reverse order to allow clean reinitialization.

4. views.sql
   - Creates SQL views to simplify queries for order summaries and inventory status.

Instructions:
-------------
1. Run drop_tables.sql first if you are resetting the database.
2. Then run create_tables.sql to create the schema.
3. Follow it with insert_data.sql to add sample data.
4. Optionally run views.sql to generate simplified queryable views.
